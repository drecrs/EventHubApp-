# EventHub Mobile Application

EventHub is a cross-platform mobile application developed using .NET MAUI and C#. The application helps event promoters advertise and manage events while allowing patrons to discover concerts, parties, sporting events, conferences, and nightlife experiences from a centralized platform.

--------------------------------------------------

FEATURES

- User Registration & Login
- Event Creation & Management
- Event Search & Filtering
- Google Maps Integration
- Event Flyer Uploads
- User Profile Management
- SQLite Local Database
- Modern Responsive UI

HOW TO RUN THE APPLICATION

1. Install Visual Studio 2022
2. Install .NET SDK
3. Install .NET MAUI Workload
4. Clone the repository
5. Open the project in Visual Studio
6. Restore NuGet packages
7. Build and run the application

--------------------------------------------------

APK BUILD

To generate the APK:

1. Open the project in Visual Studio
2. Select Release Mode
3. Right-click the Android project
4. Select Publish
5. Generate Signed APK

USER MANUAL

LOGIN
- Enter email and password

- Open Manage Events
- Enter event details

- Save event

SEARCH EVENTS
- Use search bar and category filters

OPEN LOCATIONS
- Tap location button
- Open venue using Google Maps
